Hello, I'm Damien.
I am a computer science student minoring in full-stack development.
Here's what I've done so far:
Game Project

A shoot 'em up game project for SIT102, inspired by the game Tyrian 2000.
Particle/RPi Environment Management System

A monitoring station powered by a Particle Argon that takes data from an aquarium environment and sends warnings when outside desired ranges. The Rasperberry Pi hosts a GUI control box that allows the user to set parameters.
Ping Application

This program made for Secure Networking was designed to mimic CMD's built-in ping command, allowing the user to ping an IP address and get a response.
Traceroute Application

A more advanced version of the ping app, this one takes an input IP address and determines the route taken to reach it. Mimics the built-in CMD traceroute command.
Robot Controller API

A back-end API designed to represent a Mars Rover controller.
Indigenous Gallery API

A back-end API designed to host art works and artists of First Nation's people.
